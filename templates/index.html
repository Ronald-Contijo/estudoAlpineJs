Estudo Alpine.js.

O que é Alpine.js?

Alpine.js é um pequeno framework JavaScript que oferece reatividade diretamente no HTML, de forma leve e fácil de usar. Ele funciona declarando x-data diretamente no HTML, o que cria um componente reativo com variáveis internas. Esses dados podem ser exibidos ou manipulados com outros atributos como x-text (para mostrar valores), x-show (para mostrar/ocultar elementos), ou x-on (para lidar com eventos). É como usar JavaScript diretamente no HTML, mas de forma muito mais elegante e higiênica.

x-data

O atributo x-data é parte do framework Alpine.js, que permite adicionar reatividade diretamente no HTML de forma simples e declarativa. No exemplo <h1 x-data="{ message: 'I ❤️ Alpine!! esse texto ta dentro do x-data!!' }" x-text="message"></h1>, o x-data define um pequeno componente reativo onde a variável message é inicializada com o valor 'I ❤️ Alpine!! esse texto ta dentro do x-data!!'. O atributo x-text="message" então liga o conteúdo do elemento ao valor dessa variável, fazendo com que o texto visível no <h1> seja automaticamente atualizado caso o valor de message mude. 

<h1 x-data="{ message: 'I ❤️ Alpine' }" x-text="message"></h1>
